Controls:

X - Pause Game

F - Change Skill

G - Change Weapon

Q - Dash

E - Slide

E + Space - Power jump

C - Roll

WASD - Movement

Ctrl - Crouch

Space - Jump/Vault

Mouse Left Click -  Attack

Mouse Right Click - Aim
